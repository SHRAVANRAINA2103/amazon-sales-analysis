# amazon-sales-analysis
Built an interactive Power BI dashboard to analyze Amazon sales data using KPIs, charts, and slicers. Cleaned and transformed data with Power Query and created DAX measures to track revenue, orders, and sales trends.


Amazon Sales Analysis Dashboard | Power BI

Designed and developed an interactive Power BI dashboard to analyze Amazon sales data for 2025.

Cleaned and transformed raw CSV data using Power Query, including date format correction and data type validation.

Created DAX measures for key KPIs such as Total Revenue, Total Orders, Total Quantity Sold, and Average Order Value.

Built interactive visualizations including KPI cards, bar charts, line charts, donut charts, and geographic maps.

Implemented slicers and filters for product, category, payment method, and order status to enable dynamic analysis.

Provided insights on top-performing categories, sales trends over time, payment method distribution, and customer locations.
